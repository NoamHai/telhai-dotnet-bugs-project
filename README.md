# Tel-Hai .NET Bugs Project

WPF desktop application developed as part of the **.NET course assignment (HW5)**  
at **Tel-Hai Academic College**.

The project implements a simple bug management system with categories,
demonstrating GUI development, event handling, and object-oriented design in C#.

---

## Features
- Manage bugs (add / view / update)
- Manage bug categories
- WPF-based graphical user interface
- Separation between UI and logic
- Clean object-oriented structure

---

## Tech Stack
- **C#**
- **.NET (WPF)**
- Visual Studio

---

## Project Structure
- WPF project with XAML-based UI
- Code-behind for UI logic
- Centralized data handling for bugs and categories


## How to Run
1. Open the solution file (`.sln`) in Visual Studio
2. Build the solution
3. Run the project
